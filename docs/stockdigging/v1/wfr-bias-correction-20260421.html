4. 종합 해석

4.1 두 실험이 일관되게 가리키는 결론

관점결과

품질 필터 상위 20% 결과 +2.30은 "이미 품질 좋은 종목 중에서는 TOP 20 선정이 큰 가치 없음"을, Survivorship-free -10.88은 "폐지 임박 부실주 배제 없이는 성과 음성"을 의미. 기존 +172는 "생존 2,445 종목 중 TOP 20 선정 = 폐지 임박주 자동 배제 + 품질 필터 동시 효과".

4.2 Bias 분해

기존 US-only S3 250d α = +172.07%p
├─ Survivorship bias:  약 +183%p (폐지 종목 배제 효과)
├─ 품질 필터 bias:     약 +170%p (상위 20% 밖 제거 효과)

(bias는 중첩 영역이 있어 합계가 +172를 초과)

4.3 왜 이런 결과가 나왔나?

  • 학습 기간 (2021-06 ~ 2024-12): 생존 종목 위주로 학습된 지표가 실제로는 "생존 신호"를 간접 학습
  • 폐지 임박 종목 특성: 보통 저PBR/저PER → 기존 학습에서 "저평가 매수" 신호로 잘못 선정될 가능성
  • 현재 25개 지표: fundamental + technical 조합이지만, 부실주 필터링 기능 부족 (예: 유동성, 거래량 트렌드, 신용등급 등 미포함)

5. 시사점 및 향후 방향

즉시 필요한 작업

  • 전체 S1-S5 재실행: S3 외 다른 split도 유사 패턴인지 확인 (~15h)
  • KR 같은 방식 적용: KR delisted 47개 보정 후 재학습 (단, KR은 원래 α 낮음)

근본적 재고 필요

  • 현재 25개 지표로는 순수 α 생성 불가 — 추가 지표 필요
    • 유동성 지표 (ADV, bid-ask spread)
    • 신용 위험 지표 (부채 상환 능력, Altman Z-score)
    • 내부자 거래, 공매도 비율

  • 실전 필터 포함 전략:
    • 유동성 최소 기준 (일평균 거래대금)
    • 시가총액 $1B+ + ROE > 0 + 부채비율 < 200% 등 사전 필터
    • 그 후 스코어링 기반 TOP 20 선정
  • 실전 적용 시: 기존 "+172" 같은 숫자에 의존하지 말고 실전 포트폴리오 시뮬레이션 (리밸런싱 주기, 거래비용, 폐지 처리) 필수
